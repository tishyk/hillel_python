Во вложении пример решения по спасению Пегаса который необходимо модифицировать freePegas.py.
Как?
Шаг 1. Необходимо использовать данные из конфигурационного файла game_structure.json
Шаг 2. Необходимо создать ряд функции которые описаны в файле pegas_structure.py (использовать для изменений)
    Все функции записываются в начале файле!
Шаг 3. Заменить созданными функциями строчки кода где это необходимо()
Шаг 4. Проверить что код работает
Шаг 5. Все инструкции которые находятся со строки 8 и до конца файла  pegas_structure.py
    перенести в функцию с именем main() добавив два параметра user и log_path.
Шаг 6. Создать лямбда функцию которая будет возвращать имя текущего пользователя и временную директорию пользователя.
    Словарь с пользовательскими данными можно получить из os.environ "словаря" по ключу 'USERNAME' ('noname by default')
    а временную директорию по ключу 'TEMP' (текущая директория по умолчанию).
    Лямбда функции также записываются в начале файла.
Шаг 7. Вызвать функцию main() и передать в нее значения из лямбда функции

Шаг 8*. Предоставить пользователю игры авто сохранение на случай завершения игры (break в pegas_structure.py).
 Сейчас одна неудачная попытка выбрасывает из игры. Пользователь перезапустит игру для продолжения.
 При повторном запуске игры загрузить значение необходимых переменных из сохраненного состояния если оно есть.
 Для получения всех переменных с глобальной облисти можно использовать функцию vars() или globals().
 Выбрать нужные переменные для сохранения (функция с **kwargs для фильтрации).
 Сохранить нужные переменные в файл user_name_savings.json где user_name с  ключа 'USERNAME' полученное на шаге 6.
